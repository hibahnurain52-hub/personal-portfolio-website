# Figma Desktop Design

This is a beginner-level desktop advertisement design created using Figma.

## About
This project was created as part of my UI/UX and visual design learning practice. The design helped me understand layout, spacing, typography, color balance, and how to create a clean desktop advertisement.

## Tools Used
- Figma
- UI Design
- Visual Design
- Layout Design

## What I Practiced
- Desktop ad layout
- Text placement
- Visual hierarchy
- Basic design alignment
- Color and spacing

## What I Learned

This project helped me understand how to create a clean and attractive desktop advertisement using Figma. I practiced layout design, spacing, alignment, typography, color balance, and visual hierarchy.

I also learned that good design is not only about making something look nice, but also about making the message clear and easy to understand. This practice helped me build confidence in using Figma and improved my basic understanding of UI/UX and visual design.
